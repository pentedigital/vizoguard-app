## What is Vizoguard?

Vizoguard Pro is a desktop security application for Mac and Windows that provides:

- AI-powered real-time threat blocking
- Phishing detection (8-vector URL analysis)
- Connection monitoring
- Self-healing file integrity protection
- Encrypted VPN via Shadowsocks/Outline
