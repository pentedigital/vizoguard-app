# UX Polish Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix the most visible UX friction in the Vizoguard desktop app — VPN button, settings panel, toast notifications, and instant startup.

**Architecture:** Surgical edits to 4 existing files. No new files, no refactoring. Dashboard gets settings gear + panel overlay, improved toasts, and VPN button spinner. Main process gets non-blocking startup and new IPC handlers. Preload gets new bridge methods.

**Tech Stack:** Electron, HTML/CSS/JS (no frameworks), IPC via contextBridge

**Spec:** `docs/superpowers/specs/2026-03-19-ux-polish-design.md`

**Security note:** All dynamic text inserted via `textContent` (not innerHTML) to prevent XSS. The only innerHTML usage is for the spinner element which is a static trusted string.

## Task 3: Main Process — Non-blocking Startup

**Files:**
- Modify: `main.js`

- [ ] **Step 1: Rewrite the startup flow in app.whenReady()**

Replace the current `app.whenReady().then(async () => { ... })` block with:

```js
app.whenReady().then(async () => {
  if (process.platform === "darwin") {
    app.dock.hide();
  }

  createTray(trayCallbacks);

  if (!license.hasLicense()) {
    createWindow("activate.html");
    return;
  }

  // Instant startup: load dashboard immediately from cached license
  createWindow("dashboard.html");
  updateMenu(true, trayCallbacks);

  // Background validation — don't block the UI
  license.validate().then(async (result) => {
    if (result.valid) {
      license.startPeriodicCheck();
      await startSecurityEngine();
      updateMenu(true, trayCallbacks);

      // Check for updates silently
      setTimeout(() => updater.check(), 5000);
    } else {
      // License invalid/expired/suspended — navigate away from dashboard
      showPage("expired.html");
      updateMenu(false, trayCallbacks);
    }
  }).catch((err) => {
    // Network error — log it, dashboard stays visible (grace period applies)
    console.error("Background license validation failed:", err);
  });
});
```

- [ ] **Step 2: Update the license:activate handler**

Replace the existing `ipcMain.handle("license:activate", ...)` handler with:

```js
ipcMain.handle("license:activate", async (_event, key) => {
  try {
    const result = await license.activate(key);
    // Return immediately — renderer shows success animation,
    // then calls app:showDashboard to navigate + start engine
    return result;
  } catch (err) {
    return { success: false, error: err.error || err.message || "Activation failed" };
  }
});
```

Note: The old handler called `showPage("dashboard.html")` and `startSecurityEngine()` directly. The new flow lets the renderer handle the transition via `app:showDashboard` after showing the success animation.

Replace the existing `function showToast(msg)` with:

```js
  let toastTimer = null;
  let toastIsCritical = false;

  function showToast(msg, critical) {
    critical = critical || false;
    const toast = document.getElementById('toast');
    const msgEl = document.getElementById('toast-msg');
    const dismissBtn = document.getElementById('toast-dismiss');

    // Info toasts cannot replace critical toasts
    if (!critical && toastIsCritical) return;

    // Clear any existing auto-dismiss timer
    if (toastTimer) { clearTimeout(toastTimer); toastTimer = null; }

    msgEl.textContent = msg;
    toastIsCritical = critical;

    if (critical) {
      toast.className = 'toast critical show';
      dismissBtn.style.display = '';
    } else {
      toast.className = 'toast show';
      dismissBtn.style.display = 'none';
      toastTimer = setTimeout(function() {
        toast.classList.remove('show');
        toastIsCritical = false;
      }, 3000);
    }
  }

  function dismissToast() {
    var toast = document.getElementById('toast');
    toast.classList.remove('show');
    toastIsCritical = false;
    if (toastTimer) { clearTimeout(toastTimer); toastTimer = null; }
  }
```

## Task 6: Dashboard — VPN Button Spinner + State Fix

**Files:**
- Modify: `ui/dashboard.html`

- [ ] **Step 1: Replace the toggleVpn function**

Replace the existing `async function toggleVpn()` with:

```js
  async function toggleVpn() {
    var btn = document.getElementById('vpn-btn');
    btn.disabled = true;

    if (vpnConnected) {
      btn.textContent = 'Disconnecting...';
      var result = await window.vizoguard.vpnDisconnect();
      if (result.success) {
        vpnConnected = false;
        updateVpnUI(false);
        showToast('VPN disconnected');
      } else {
        showToast('Disconnect failed: ' + result.error, true);
      }
    } else {
      btn.textContent = 'Connecting...';

      // Ensure we have a VPN key first
      var keyResult = await window.vizoguard.getVpnKey();
      if (keyResult.error) {
        showToast('No VPN key: ' + keyResult.error, true);
        btn.disabled = false;
        btn.textContent = 'Connect';
        return;
      }

      var connectResult = await window.vizoguard.vpnConnect();
      if (connectResult.success) {
        vpnConnected = true;
        updateVpnUI(true);
        showToast('VPN connected');
      } else {
        showToast('Connect failed: ' + (connectResult.error || 'Unknown error'), true);
      }
    }

    btn.disabled = false;
  }
```

  async function toggleSettings() {

    if (isOpen) {
